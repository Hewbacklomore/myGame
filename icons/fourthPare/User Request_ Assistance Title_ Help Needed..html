Здається, ви хочете створити простий калькулятор в JavaScript і запитати користувача, яку операцію він хоче виконати. Для цього можна використати такий код:

let operation = prompt("Яку операцію ви хочете виконати? Введіть 'add', 'sub', 'mult' або 'div':");

if (operation === "add") {
  // Додавання
  let num1 = prompt("Введіть перше число:");
  let num2 = prompt("Введіть друге число:");
  let result = parseFloat(num1) + parseFloat(num2);
  alert(`Результат додавання: ${result}`);
} else if (operation === "sub") {
  // Віднімання
  let num1 = prompt("Введіть перше число:");
  let num2 = prompt("Введіть друге число:");
  let result = parseFloat(num1) - parseFloat(num2);
  alert(`Результат віднімання: ${result}`);
} else if (operation === "mult") {
  // Множення
  let num1 = prompt("Введіть перше число:");
  let num2 = prompt("Введіть друге число:");
  let result = parseFloat(num1) * parseFloat(num2);
  alert(`Результат множення: ${result}`);
} else if (operation === "div") {
  // Ділення
  let num1 = prompt("Введіть перше число:");
  let num2 = prompt("Введіть друге число:");
  let result = parseFloat(num1) / parseFloat(num2);
  alert(`Результат ділення: ${result}`);
} else {
  alert("Невідома операція. Спробуйте ще раз.");
}

Цей код використовує функцію prompt, щоб запитати користувача, яку операцію він хоче виконати. Потім він використовує різні блоки if / else if, щоб виконати правильну операцію залежно від введеного користувачем значення. Код перевіряє, чи введене значення є одним з чотирьох дозволених варіантів: 'add', 'sub', 'mult' або 'div'. Якщо ні, він повідомляє користувача про помилку.

Отже, питання про те, що чекає на людину після смерті, є складним та спірним, і відповідь на це питання залежить від релігійних, культурних та наукових переконань кожної людини.

Прямі докази існування НЛО

Насправді немає прямих доказів існування НЛО. Термін "НЛО" (Незвідане Літальне Об'єкт) означає будь-який об'єкт, який не можна ідентифікувати як природний чи штучний. Таким чином, будь-який об'єкт, який не може бути ідентифікований, може бути класифікований як НЛО.

Багато звітів про НЛО може бути пояснено як природні феномени, такі як метеори, плазма, світло від маяків чи повітряних суден. Також, багато спостережень НЛО може бути пояснено як військові та комерційні повітряні судна.

Хоча деякі спостереження НЛО залишаються не поясненими, це не означає, що це прямі докази існування позаземного життя або інших надприродних явищ. Для прийняття таких висновків потрібні науково-доведені докази, які підтримуються емпіричними даними та спостереженнями.
